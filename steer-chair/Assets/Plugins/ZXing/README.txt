ZXing.Net for QR Code Scanning
==============================

EASY SETUP (Recommended)
------------------------
In Unity Editor:
  1. Go to menu: Sensocto > Auth > Setup QR Scanner
  2. Click "Download" when prompted
  3. Done! The library will be downloaded and configured automatically.

MANUAL SETUP
------------
If automatic download fails:
  1. Download ZXing.Net from: https://github.com/micjahn/ZXing.Net/releases
  2. Extract zxing.dll from lib/unity/ or lib/net20/ folder
  3. Place it in this folder (Assets/Plugins/ZXing/)
  4. Unity will automatically detect it and enable QR scanning

CHECK STATUS
------------
Menu: Sensocto > Auth > Check QR Scanner Status
This will show whether ZXing is installed and configured correctly.
